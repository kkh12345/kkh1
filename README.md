
#  공공기관리뉴얼 포트폴리오 페이지입니다.


메인 페이지 기능
  * 스와이퍼
  * 전국지부 바로가기 ) pc : 모달창 , 태블릿,모바일사이즈 : 드롭다운
  * 네비게이션 서브메뉴 ) pc : 드롭다운 태블릿,모바일 : 햄버거 메뉴로 변경,클릭시 사이드바(아코디언)
