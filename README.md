##### 주요 기능:
- 스크롤 시 헤더 고정
- 검색아이콘 클릭 시 검색 모달(PC)
- 네비게이션 메뉴 호버 시 서브메뉴 드롭다운(PC)
- 서브메뉴 중 공지사항 클릭 시 공지사항 페이지로 이동
- 햄버거 메뉴 클릭 시 사이드 네비게이션 등장(모바일, 태블릿)
